此域｜報稅記帳簿

同一筆交易，同時保留外帳申報欄位與內帳經營標籤。可用 Supabase 讓手機與電腦同步。

商號資料

外帳用於稅務與會計師整理；內帳用於實際經營分析。兩者不得做成互相矛盾的兩套帳。

手機／電腦雲端同步

使用 Supabase Auth、Database、Storage 同步流水帳與附件。GitHub 仍只放程式碼，不保存你的記帳資料。

未設定

第一次使用需先在 Supabase 建立資料表、Storage bucket 與安全規則。請照 README 的 SQL 設定。

流水帳

點選「檢視憑證」可開啟存在本機或雲端同步回來的附件。

此工具僅作記帳與憑證整理，不取代會計師或國稅局核定。外帳應以真實交易與合法憑證為準。
